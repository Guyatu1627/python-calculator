# Python Calculator

## Description
This is a simple Python calculator project designed for beginners. The calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It serves as an excellent introduction to programming in Python and can be expanded with additional features and functionality.

## Features
- Perform basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Simple and user-friendly interface.
- Code structure suitable for beginners.
